PLAKAR-LOGOUT(1) - General Commands Manual

# NAME

**plakar-logout** - Log out from Plakar services

# SYNOPSIS

**plakar&nbsp;logout**

# DESCRIPTION

The
**plakar logout**
command logs out an authenticated session with the Plakar platform.

# EXAMPLES

Log out from the current session:

	$ plakar logout
